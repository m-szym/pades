\doxysubsection{Detailed Description}
Component for loading files with a specific extension filter and verification. 

This class provides a UI component for selecting files, verifying them, and displaying the status. Uses the provided File\+Load\+Tester to validate the selected file. If the file is valid, it updates the label to show the file name and a success message and provides the file with get\+File(). If the file is invalid, it shows an error message and resets the file selection.
